# IT Infrastructure Home Lab

A hands-on IT Support, Help Desk, and System Administration lab built using VirtualBox, Windows 10, and Ubuntu Linux.

## Project Objectives

- Build and manage Windows and Linux virtual machines
- Practice networking, user administration, and system administration tasks
- Develop troubleshooting and documentation skills
- Simulate real-world IT Support and Help Desk responsibilities
